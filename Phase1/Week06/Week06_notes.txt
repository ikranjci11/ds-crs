1. Describe what you saw in the Gender column from value_counts(). Why is this kind of inconsistency a problem for analysis, and how would you approach cleaning it?

I saw that people used different values to give the same answer (e.g. male, MALE, M, m...). This is a problem because pandas would treat all of these answers as different, when in reality they have the same meanings. Also a few answered neither male nor female, with many different other answers.
First of all, I would change all the data that shares the same meaning into one consistent value. For example, M for all males. All the answers that are neither M nor F I would code as 'Other'.
