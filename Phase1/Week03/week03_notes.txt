LISTS

- we are using [] to store values in a list, unlike variables we are storing multiple items

friends = ["Kevin", "Karen", "Jim"] -> it can also contain numbers and bullions

print(friends)

print(friends[0]) -> prints Kevin; we can specify which element from the list we want
** -1 starts indexing from the back of the list

print(friends[1:]) -> from element 1 onwards

print(friends[1:3]) -> grabs 1 and 2, but not 3!!


friends[1] = "Mike" -> modifies the list, but not permanently?

print(friends[1]) -> prints Mike now

FUNCTIONS WITH LISTS IN PYTHON

lucky_numbers = [4, 8, 15, 16, 23, 42]
friends = ["Kevin", "Karen", "Jim", "Oscar", "Toby"]

friends.extend(lucky_numbers) -> adds another list to the first one
friends.append("Creed") -> adds an element at the end
friends.insert(1, "Kelly") -> adds an element into a specific spot
friends.remove("Jim")
friends.clear() -> empties the list
friends.pop() -> removes the last element

print(friends.index("Kevin")) -> returns the index of the element
print(friends.count("Jim"))  -> counts the number of elements

friends.sort() -> alphabetically or by number size
lucky_numbers.reverse() -> reverses the order of the list
friends2 = friends.copy() -> copies a list into a new one


FOR LOOPS

- looping through arrays, letters of a string, numbers and so on

for letter in "Giraffe Academy":
	print(letter)

friends = ["Kevin", "Karen", "Jim", "Oscar", "Toby"]

for friend in friends:
	print(friend)

for index in range(10):
	print(index)     -> prints all the numbers 0-9, not including 10!

for index in range(3, 10):
	print(index)      -> 3-9


len(friends) -> 3

for index in range(len(friends)):
